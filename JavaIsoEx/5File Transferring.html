File Transferring

Right click on the person you want to send a file to, an option button will pop-up.
Click on the transfer file button and select the file that you want to share.

If you receive a file there will be a pop-up that tells you that this ID sent you a file.
You can choose whether to accept or not, if accept you can choose the location where will the file to be downloaded.

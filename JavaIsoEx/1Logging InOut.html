How to Login

First, run the program and you will see the login screen. You can login as an Inhabitant if you have a username and password.
If you don't have one, you can login as an Alien instead.
(Note that if you login as an Alien, you will not be able to get full function of the program)

Login Functions

This login will remember your username everytime you have logged in to the ICE World.
If you double click on the username, that username will appear on the username.
